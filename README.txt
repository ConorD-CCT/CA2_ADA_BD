There are two notebooks in this folder

Election Tweets-MongoDB extra.ipynb:
This file performs the extraction of the data from the mongo database and performs the sentiment scoring and groups by day. The output from this is daily_sentiment_pd.

Sentiment Analysis and Forecasting.ipynb:
This file imports the data from daily_sentiment_pd, analyses it further, performs the forecasting, and makes an interactive dashboard to present the results.

The two benchmarking files are results from the benchmarking study between mySQL and mongoDB.

daily_sentiment_pd is the output from Election Tweets-MongoDB extra.ipynb
